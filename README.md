# morse

Small module for converting between English and Morse code.

## Usage

    Morse.translate("github")
    #=> "--. .. - .... ..- -..."

    Morse.translate(".. / .-.. --- ...- . / -- --- .-. ... . / -.-. --- -.. .")
    #=> "i love morse code"
